Command Line Cheat Sheet

Command Meaning Use
Directory Folder Used when describing the folder structure of a project
↑ ↓ buttons Cyles through previous commands
tab button Auto-completes command in possible
ls List List the files and folder within the directory
pwd Print Working Directory Lists out the full route of the folder
cd Change Directory Moves back or forward through the folder tree
mkdir Make Directory Creates a new folder
touch Creates a new file
clear Clears previous commands from view
